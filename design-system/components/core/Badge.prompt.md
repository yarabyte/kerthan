Compact pill for statuses, categories and service tags.

```jsx
<Badge tone="green" dot>Ouvert 24h/24</Badge>
<Badge tone="gold">Nouveau</Badge>
<Badge tone="neutral" leftIcon={<Icon name="clock" size={13} />}>Lun–Sam</Badge>
```

Tones: `green` `gold` `red` `neutral` `solid-green` `solid-gold`. Use `dot` for a live status indicator.
