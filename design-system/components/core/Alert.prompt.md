Inline status banner.

```jsx
<Alert tone="success" title="Rendez-vous confirmé">
  Nous vous attendons le 12 juin à 09h00.
</Alert>
<Alert tone="warning">Les urgences sont prioritaires sur les rendez-vous.</Alert>
```

Tones: `success` `info` `warning` `danger`. Optional `title`; `icon` defaults on.
