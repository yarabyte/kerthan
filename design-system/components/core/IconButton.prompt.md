Circular icon-only button for toolbars, social links, and compact actions.

```jsx
<IconButton icon="phone" variant="solid" label="Appeler" />
<IconButton icon="facebook" variant="outline" label="Facebook" />
<IconButton icon="menu" variant="ghost" size="lg" label="Menu" />
```

Variants: `solid` (green), `gold`, `outline`, `ghost`. Sizes `sm` `md` `lg`. Always pass `label`.
