Soft surface container — the base for most content blocks.

```jsx
<Card hover accent="green">
  <h3>Médecine Interne</h3>
  <p>Prévention et traitement des maladies de l'adulte.</p>
</Card>

<Card variant="brand">…deep-green promo panel…</Card>
```

Props: `pad` (default true), `hover` (lift), `accent` ("green"|"gold" top border), `variant` ("default"|"brand"|"flat").
